# Impostor

**Impostor** is a multiplayer social deduction game where players join a lobby, chat, and try to uncover the impostor. The game is fully playable, self-contained, and will not receive further updates.

---

## How to Play

1. **Join the Game**
   - Go to `impostor-7mgm.onrender.com`.
   - Enter a unique name and click "Join".
   - Wait for other players to join the lobby (minimum 2 players required).

2. **Start the Game**
   - Once enough players have joined, the "Start Game" button will become active.
   - Click it to begin the game.

3. **Roles**
   - Each player is assigned a role:
     - **Impostor:** Try to blend in and avoid being discovered.
     - **Innocent:** Work together to find the impostor by discussing clues.

4. **Chat and Voting**
   - Use the chat to discuss and gather information.
   - When ready, vote using the command:
     ```
     /vote [player_name]
     ```
   - You cannot vote for yourself.
   - Once all players have voted, the game will reveal the results.

5. **Game End**
   - If the impostor is voted out, innocents win.
   - If the impostor survives, the game ends and the impostor's role is revealed.
   - After the game, players return to the lobby to start a new round.

---

## Controls

- **Chat:** Type messages or commands in the chat input box.
- **Voting:** Use `/vote [player_name]` to cast your vote.
- **Role Reminder:** You can view your role again anytime during a game by clicking "Show My Role Again".

---

## Notes

- The game is self-contained and requires no additional setup beyond running the server.
